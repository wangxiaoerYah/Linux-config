# Tip
Hooks目录:
Archlinux: ` /etc/pacman.d/hooks `

` pacman可以在处理前后，运行/usr/share/libalpm/hooks/文件夹下的hooks，更多的hooks文件夹可以通过HooDir选项在pacman.conf中指明，默认/etc/pacman.d/hooks。`
